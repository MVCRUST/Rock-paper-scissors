# Rock-paper-scissors
Leanne and Max attempt to create a Rock, paper, scissors game

**Instructions**
- Computer and user player, each will generate a random number (0, 1, or 2)
- Each of the above numbers will correspond with Rock, paper, or Scissors
- Using if statements, we will create win conditions for each scenario.
- Then implement score for user and computer, winner will make score increase by one.
- A draw must have no effect on the score.
- Once either user or computer reaches 3, both scores reset.
